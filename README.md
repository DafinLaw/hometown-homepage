This is the completed Hometown Homepage from Scrimba (https://scrimba.com/fullstack-path-c0fullstack/~040)
The technologies used are HTML and CSS
The project consists of 3 sections targeting 3 main areas. The first is the hero section and it displays an overview and the name of the town. 
The second is the activities section and it displays 3 activities usually done by the inhabitants. The last section displays the guide in this
town.
